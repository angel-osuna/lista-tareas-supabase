# 📝 Aplicación de Lista de Tareas con Supabase

Este proyecto es una aplicación web sencilla e interactiva de gestión de tareas (To-Do List) desarrollada como actividad universitaria. El objetivo principal es demostrar la integración de un frontend ligero con una base de datos real en la nube utilizando operaciones CRUD (Crear, Leer, Actualizar, Eliminar).

## 🚀 Características

* **Agregar tareas:** Permite registrar nuevas actividades en la base de datos en tiempo real.
* **Marcar como completada:** Sistema interactivo para tachar tareas terminadas y actualizar su estado.
* **Eliminar tareas:** Borrado definitivo de los registros en la base de datos.
* **Interfaz responsiva:** Diseño moderno y adaptable construido rápidamente con Tailwind CSS.

## 🛠️ Tecnologías Utilizadas

**Frontend:**
* HTML5
* JavaScript (Vanilla JS)
* Tailwind CSS (vía CDN)

**Backend / Base de Datos (BaaS):**
* [Supabase](https://supabase.com/) (PostgreSQL)

## ⚙️ Instalación y Uso

Al ser un proyecto basado en Vanilla JavaScript y HTML estático, no requiere la instalación de dependencias locales ni servidores complejos (como Node.js o XAMPP).

**Para probarlo localmente:**
1. Clona este repositorio o descarga los archivos en un `.zip`.
2. Abre el archivo `index.html` directamente en tu navegador web de preferencia.

**Para verlo en vivo:**
* El proyecto está desplegado y funcionando a través de GitHub Pages. Puedes acceder al enlace en la sección "Environments" o "About" de este repositorio.
